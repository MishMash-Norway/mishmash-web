## Author affiliations

Since MishMash is not an organizational entity, authors should not use MishMash affiliation in publications. Instead, they should use their regular institutional attribution and add a funding acknowledgment (see below).

## Paper acknowledgments

Use the following attribution text in papers and related publications:

> This research was supported by the Research Council of Norway through the MishMash Centre for AI and Creativity, Project Number 357438.

## NVA registrations

For researchers reporting using NVA (the Norwegian Research Information Repository), all research results should be registered using the [MishMash project ID in NVA](https://nva.sikt.no/projects/2744839) (Project ID 2744839). Note that the NVA ID links to the RCN project number, making it easy to report correctly to RCN.
