## Summary
This two-day hackathon at NTNU explores AI-based pose detection (such as Google’s MediaPipe) as a foundation for movement-to-music interaction in therapeutic and wellbeing settings. Researchers, artists, and students collaboratively prototype mapping strategies between body movement and musical parameters, evaluating how affordable markerless tracking compares to established computer vision techniques. The project produces an open-source code repository and documentation to support future large-scale funding applications for AI-assisted therapeutic interactions.
