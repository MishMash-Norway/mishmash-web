## Summary
Using screenwriter Ole Christian Solbakken’s screenplay *Vita Vigeland* as a case study, this project examines how generative AI can be integrated into contemporary film preproduction. The research tests how AI tools can prototype visual dramaturgy, tone, and atmosphere for a biographical drama while reflecting on artistic control and authorships. The project contributes to Solbakken’s ongoing PhD research, documenting hybrid workflows that combine traditional film practice with emerging AI technologies.
